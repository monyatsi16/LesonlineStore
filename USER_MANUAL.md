# LesOnline User Manual

## Introduction
LesOnline is a smart marketplace and dynamic pricing platform built for Lesotho. It serves three main user roles:
- **Buyers**: Browse products, add items to cart, and place orders.
- **Retailers (sellers)**: Manage products, view orders, review pricing recommendations, and monitor sales performance.
- **Administrators**: Manage the entire platform, users, products, orders, analytics, and automated pricing operations.

All prices are displayed in Lesotho Loti (LSL/M).

## Quick Start

### System Requirements
- Node.js 20 or later
- PostgreSQL-compatible database (Neon serverless is used by default)
- npm package manager

### Install Dependencies
```bash
npm install
```

### Run Locally
Start the backend server:
```bash
npm run dev
```

Start the client app:
```bash
npm run dev:client
```

### Seed the Database
Populate the database with sample products and admin/seller accounts:
```bash
npm run seed
```

## User Roles and Accounts

### Buyer
- No login required
- Can browse the marketplace, view products, add items to cart, and checkout

### Seller / Retailer
- Manage own product catalog
- Access pricing recommendations and analytics
- Review and apply price updates

### Admin
- Full platform control
- Manage users, products, orders, and pricing operations
- View overall analytics and pricing update history

## Buyer Guide

### Browse the Marketplace
- Visit the homepage to see featured and trending products.
- Use search and category filters to quickly find items.
- View product details to see pricing information, product description, and order options.

### Add to Cart
- On a product page, choose quantity and add items to the cart.
- Open the cart page to review selected items.
- Adjust quantities or remove items before checkout.

### Checkout
- Complete the buyer order form with delivery and contact details.
- Submit the order to reserve inventory and record the purchase.

## Seller Guide

### Login and Dashboard
- Sellers access the `/auth` page to sign in.
- After login, visit the `/dashboard` page for an overview of products, orders, and analytics.

### Managing Products
- View your product list and key details such as price, stock, and sales.
- Update product prices manually when needed.
- Review pricing recommendations generated by the model.

### Pricing Recommendations
- The platform uses a real gradient boosting pricing model to suggest smart prices.
- Sellers can review and apply recommendations for each product.
- The model considers market position, brand, stock, demand, and category trends.

### Sales Analytics
- Use charts and reports to track sales performance.
- Monitor orders received, revenue, and inventory impact.

## Admin Guide

### Admin Dashboard
- Login to the admin area to access platform-wide management tools.
- Use the admin dashboard to monitor business metrics and overall performance.

### User Management
- View all users and their roles.
- Change user roles between admin, retailer, and buyer where applicable.

### Product and Order Management
- Browse all marketplace products and remove items if necessary.
- Review all orders across the platform.

### Pricing Operations
- Run the pricing model across all products from the admin controls.
- Review pricing update history and scheduler status.
- Manually trigger price updates when needed.

## Pricing Model Overview

LesOnline features a brand-aware gradient boosting pricing engine that:
- Analyzes product price position within categories
- Uses brand and category performance data
- Considers stock, views, demand, and conversion metrics
- Generates recommended prices and confidence scores

The pricing scheduler can automatically update prices on a recurring cycle.

## Key Features
- Public marketplace with search, categories, and product details
- Buyer cart and checkout flow
- Seller product management and pricing recommendations
- Admin platform control with analytics and pricing logs
- Real dynamic pricing model with brand-awareness
- Role-based authentication and secure admin/seller routes

## Useful Commands

## Notes
- The system is designed for Lesotho marketplace operations and prices in LSL.
- Admin and seller interfaces are separated from the buyer marketplace for clarity.
- Pricing updates are logged and controlled through the admin panel.
